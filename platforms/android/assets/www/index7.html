Для проведения расчётов Разности энтальпий необходимо:
1. Перейти в раздел Разность энтальпий.
2. Выбрать необходимый продукт из списка.
3. При наличии продукту в списке необходимо заполнить поля Начальная и Конечная температура. При заполнении следует придерживаться ограничении.
4. После заполнения полей, необходимо нажать кнопку Расcчитать разность Энтальпий. В полях расположенных ниже выводятся название продукта и результат вычислений.
5. В случае если продукт отсутствует, выбрать соотвествующую графу:  Нет в списке или неизвестен
6. Для заполнения добавятся ещё два новых поля: Криоскопическая температура и Содержание воды.
7. Расчёты осуществляются по формулам Рауля или Чижова.
8. После заполнения полей, необходимо нажать кнопку Расcчитать разность Энтальпий. В полях расположенных ниже выводится результат вычислений.
9. После осуществления расчётов в нижней части экрана можно рассмотреть график зависимости энтальпии продукта от температуры.
Для проведения расчётов Влагосодержания продукта необходимо:
1. Перейти в раздел Влагосодержание продукта.
2. Выбрать из списка необходимые величины.
3. Заполнить поля выбранных ранее величин соответствующими коэффициентами. При заполнении следует придерживаться ограничении.
4. После заполнения полей, необходимо нажать кнопку Расcчитать.
 - Если поля заполнены верно, то вы получите название величин и, соответственно, результаты вычислений.
 - Если поля были заполнены неверно или ошибка связанна с выбраной величиной, то рядом с полем, в котором была допущенна ошибка, появиться сообщение ошибки.
 - Если значение не входило в допустимые, оно изменяется на ближайшее допустимое, а рядом с полем, в котором была допущенна ошибка, будет выведенно ближайшее доступное чиcло.
Для перехода между модулями можно воспользоваться главной страницей или переходом по нажатию на название модуля, который находится в верхней части каждого модуля.
